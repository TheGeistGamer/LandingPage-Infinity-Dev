# InfinityDev Agency Landing Page

Official landing page for InfinityDev Agency, a software development company focused on creating modern web applications, mobile apps, and scalable digital solutions.

## 🚀 About The Project

This project is a modern, responsive landing page built to showcase InfinityDev's services, portfolio, technologies, and expertise.

The website aims to provide businesses and startups with a clear overview of our development capabilities and how we help transform ideas into successful digital products.

### Key Features

* Modern and responsive design
* Fast performance and SEO optimization
* Service showcase sections
* Portfolio and projects display
* Contact and lead generation forms
* Mobile-first experience
* Smooth animations and transitions
* Scalable Astro architecture

## 🧞 Commands

All commands are executed from the root directory.

| Command                   | Description                      |
| ------------------------- | -------------------------------- |
| `npm install`             | Install project dependencies     |
| `npm run dev`             | Start development server         |
| `npm run build`           | Generate production build        |
| `npm run preview`         | Preview production build locally |
| `npm run astro ...`       | Execute Astro CLI commands       |
| `npm run astro -- --help` | Display Astro CLI help           |

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/infinitydev-agency.git
```

Navigate to the project:

```bash
cd infinitydev-agency
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:4321
```
